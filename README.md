# VScan-bate

![VScan Pro Logo](https://github.com/ryan611ryan-arch/Vscan-beta/blob/main/image.jpg)

Advanced Port and Vulnerability Scanner using Nmap

## Features

- Fast port discovery using SYN scan
- Deep service detection with version fingerprinting
- OS fingerprinting
- Vulnerability assessment for common services
- Nmap script integration
- Professional formatted output

## Requirements

- Python 3.6+
- Nmap installed on system
- Python packages: python-nmap, colorama

## Installation

```bash
git clone [https://github.com/ryan611ryan-arch/Vscan-beta.git](https://github.com/ryan611ryan-arch/Vscan-beta.git)
cd Vscan-beta
pip install -r requirements.txt
sudo python3 Vscan.py
