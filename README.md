# VScan Pro

Advanced Port and Vulnerability Scanner using Nmap

## Features

- Fast port discovery using SYN scan
- Deep service detection with version fingerprinting
- OS fingerprinting
- Vulnerability assessment for common services
- Nmap script integration
- Professional formatted output
- No firewall detection scripts

## Requirements

- Python 3.6+
- Nmap installed on system
- Python packages: python-nmap, colorama

## Installation

```bash
git clone https://github.com/YOUR_USERNAME/vscan.git
cd vscan
pip install -r requirements.txt
